# **App Name**: Renda Online Max

## Core Features:

- Affiliate Marketing Module: Allows users to browse and select affiliate marketing products from various vendors.
- E-commerce Sales Dashboard: Dashboard showing the status of products being sold online
- Digital Investment Insights: Provides insights into digital investment opportunities and trends, curated by a generative AI tool to enhance financial decision-making.
- Tutorial and Guide Library: A curated library of video tutorials and guides focused on generating online income, with categories like 'Marketing Digital' and 'Vendas Online'.
- Product Comparison Tool: Compare products based on reviews using AI and summaries, and assist users in determining products to purchase.

## Style Guidelines:

- Primary color: Vibrant green (#7CFC00) to represent growth and financial success.
- Background color: Light pastel green (#F0FFF0).
- Accent color: Blue (#6495ED) for interactive elements and call-to-action buttons.
- Body and headline font: 'Inter', sans-serif, for a modern, machined, neutral look that provides clarity and readability.
- Use modern and minimalist icons to represent different categories and features within the platform.
- Maintain a clean and organized layout with clear sections for easy navigation.
- Subtle transitions and animations to provide visual feedback and enhance the user experience.
